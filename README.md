# spotify-data-analysis
This project analyzes Spotify tracks dataset to discover patterns in genres, popularity, audio features, and trends over time.

#Tools Used:
- Python
- Pandas / NumPy
- Matplotlib / Seaborn
- Google Colab

DATASETS USED:

SpotifyFeatures.csv
https://www.kaggle.com/datasets/zaheenhamidani/ultimate-spotify-tracks-db

tracks.csv
https://www.kaggle.com/datasets/lehaknarnauli/spotify-datasets
